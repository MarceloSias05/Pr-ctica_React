# React Unity Game Application

## Repository Information
- Project: React Unity WebGL Integration
- Author: Marcelo Sias
- Technology Stack: React + Unity WebGL + Express.js

## Setup Instructions

1. Install dependencies:
   npm install

2. Unity Game Setup:
   - Place your Unity WebGL build files in: public/unity/game/Build/
   - Required files:
     * game.loader.js âœ… (already present)
     * game.data.br (compressed game data)
     * game.framework.js.br (compressed framework)
     * game.wasm.br (compressed WebAssembly)

3. Run the application:
   npm run dev-with-unity
